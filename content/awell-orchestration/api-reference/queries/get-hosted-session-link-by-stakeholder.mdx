---
title: Get Hosted Session Link
description: Get Hosted Session Link for a given stakeholder
---

This request allows us to fetch the hosted session link for a given stakeholder.


## Request

### Query

```graphql
query GetHostedPagesLink($pathwayId: String!, $stakeholderId: String!) {
  hostedPagesLink(pathway_id: $pathwayId, stakeholder_id: $stakeholderId) {
    code
    success
    hosted_pages_link {
      url
      __typename
    }
    __typename
  }
}
```

### Variables

```json
{
  "pathwayId": "{{PATHWAY_ID}}",
  "stakeholderId": "{{STAKEHOLDER_ID}}"
}
```

## Response

The response object shape is the same for both queries.

```json
{
  "data": {
    "hostedPagesLink": {
      "code": "200",
      "success": true,
      "hosted_pages_link": {
        "url": "https://goto.development.awell.health/l/someid",
        "__typename": "HostedPagesLink"
      },
      "__typename": "HostedPagesLinkPayload"
    }
  }
}
```
