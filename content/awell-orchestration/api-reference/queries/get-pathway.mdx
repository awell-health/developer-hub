---
title: Get pathway
description: Get all information for a given pathway
---

> We suggest reading the page on our [domain model](/awell-orchestration/docs/getting-started/domain-model) first to get an understanding of the pathway model.

## Request

### Query

```graphql
query GetPathway($pathway_id: String!) {
  pathway(id: $pathway_id) {
    pathway {
      id
      title
      pathway_definition_id
      patient_id
      patient {
        id
      }
      latestActivities {
        id
        stream_id
        date
        action
        status
        resolution
        reference_id
        container_name
        isUserActivity
        public
        object {
          id
        }
        indirect_object {
          id
        }
        track {
          id
        }
        label {
          id
        }
        sub_activities {
          id
        }
        context {
          instance_id
        }
      }
      version # version number
      release_id # unique id representing the version
      status
      status_explanation
      start_date
      complete_date # if pathway is completed
      stop_date # if pathway is stopped
    }
  }
}
```

### Variables

```json
{
  "pathway_id": "{{PATHWAY_ID}}"
}
```
