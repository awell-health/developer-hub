## Webhooks & events

You can leverage webhooks to track events happening within Awell and send them to any destination (Segment, Amplitude, Google Analytics, ...).

## Connect to Awell's data repository

We use Big Query as our data repository to store, index, and search pathway data sets. You can either **connect your own BI tool** to our data repository in Elastic or **use our Big Query Studio to explore and visualise data**.
