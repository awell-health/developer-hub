Awell's server-side helper libraries (also known as server-side SDKs) reduce the amount of work required to use Awell's GraphQL APIs, starting with reducing the boilerplate code you have to write. There is currently only a Node.js library available.

## Installation

```bash
npm install @awell-health/awell-sdk
# or
yarn add @awell-health/awell-sdk
```

To learn more about how to use the SDK, read the [README file](https://www.npmjs.com/package/@awell-health/awell-sdk).
